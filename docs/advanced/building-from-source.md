## LinBPQ

LinBPQ source is available from G8BPQ:

```bash
git clone https://github.com/g8bpq/LinBPQ.git
cd LinBPQ
make
```

### Dependencies

```bash
sudo apt-get install build-essential libpcap-dev libasound2-dev
```
