# Contributing

!!! note "Documentation In Progress"

## How to Contribute

We welcome contributions! Here's how to help:

### Documentation

- Fix typos or unclear instructions
- Add missing content
- Improve examples

### Code

- Bug fixes
- New features
- Script improvements

### Hardware

- STL improvements
- New radio mount designs
- Enclosure variants

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Repository

- [GitHub: dpaschal/tarpn-node](https://github.com/dpaschal/tarpn-node)

## Code of Conduct

Be respectful and constructive. We're all here to improve amateur radio.
