!!! note "Documentation In Progress"

## Overview

Pre-built SD card images simplify TARPN deployment. This guide covers creating custom images.

## Tools

- [pi-gen](https://github.com/RPi-Distro/pi-gen) - Official Raspberry Pi image builder
- [Packer](https://www.packer.io/) - Automated image building

## Basic Process

1. Start with Raspberry Pi OS Lite
2. Run TARPN installer
3. Configure base settings
4. Shrink and compress image
